SSH COPY ID

Berguna untuk remote SSH tanpa perlu Insert password, sebab password sudah tersimpan.

Hasilkan SSH key
Dengan OpenSSH, SSH key dibuat menggunakan ssh-keygen. Dalam bentuk yang paling sederhana, jalankan ssh-keygen dan jawab pertanyaannya. Seperti berikut :

Membuat pasangan kunci ( kunci publik dan kunci pribadi ) hanya membutuhkan satu menit. File-file kunci biasanya disimpan di direktori ~ / .ssh.

Copy the key to a server
Setelah SSH key dibuat, perintah ssh-copy-id dapat digunakan untuk menginstalnya sebagai kunci resmi di server. Setelah kunci disahkan untuk SSH, kunci itu memberikan akses ke server tanpa kata sandi.

Gunakan perintah berikut untuk copy SSH key:

ssh-copy-id -i ~/.ssh/mykey user@host

Hanya public key yang disalin ke server. private key tidak boleh disalin ke komputer lain.

Test key nya
setelah key nya di copy, ada baiknya di test :

ssh -i ~/.ssh/mykey user@host

sekarang login seharusnya tidak menanyakan password.
